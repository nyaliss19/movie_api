This is the documentation for an API called myFlix.

Endpoints

/ returns a default landing page for myFlix

/movies returns a list of all movies in the API

/movies/:title returns information about specific movie

/genre/:Name returns specific genre

/directors/:Name returns specific director

/users post new user

/users search for specific user

/users/:username returns a specific user

/users/:username update user info by username

/users/:Username/movies/:MovieID add new movie to users array

/users/:Username/movies/:MovieID delete movie from users array

/users/:username delete user by username
