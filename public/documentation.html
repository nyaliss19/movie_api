This is the documentation for an API called myFlix.

Endpoints

/movies returns a list of all movies in the API

/movies/:title returns information about specific movie

/movies/genre/:genreName returns specific genre

/movies/directors/:directorName returns specific director

/users post new user

/users/:id change user id

/users/:id/movieTitle add new movie to users array

/users/:id/movieTitle delete movie from users array

/users/:id delete user
