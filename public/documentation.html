myFlix API

This is an API that stores movies and information on said movies for external use.

Business Logic URL HTTP Method Query parameters Request body data format Response body data format

              "Name": "Jonathan Demme",
              "Bio": "Robert Jonathan Demme was an American director, producer, and screenwriter.",
              "Birth": "1944",

          "ImagePath": "silenceofthelambs.png",
          "Featured": true
        }, 
        ...
      ]

Return data about a genre (description) by name/title (e.g., “Thriller”) /movies/genre/[Genre] GET :Genre None A JSON object holding data about the selected genre. Example: 
              
    { 
      "Name": "Thriller",
      "Description": "Thriller film, also known as suspense film or suspense thriller, is a broad film genre that involves excitement and suspense in the audience. Tension is created by delaying what the audience sees as inevitable, and is built through situations that are menacing or where escape seems impossible." 
    }
Return data about a director (bio, birth year, death year) by name /movies/directors/[Name] GET :Name None A JSON object holding data about the selected director. Example: 
              
    { 
        "Name": "Denis Villeneuve",
        "Bio": "Denis Villeneuve is an award winning director.",
        "Birth": "1967"
    }
Allow new users to register /users POST None A JSON object holding data about the user to add. Example: 
              
    { 
      "Username": "Coheed",
      "Password": "the1&only", 
      "Email": "coheed@mail.com", 
      "Birthday": "1975-03-04" 
    }
A JSON object holding data about added user including ID. Example: 
              
    { 
      "Username": "Coheed",
      "Password": "the1&only", 
      "Email": "coheed@mail.com",
      "Birthday": "1975-03-04T00:00:00.000Z",
      "FavoriteMovies": [],
      "_id": "62d457b759eb3267c6b240dc"
    }
Return user info by username /users/[Username] GET :Username None A JSON object holding data about one user. Example: 
                         
    {
      "_id": "62b5176a8ee3f54638b26e4c",
      "Username": "Coheed",
      "Password": "the1&only", 
      "Email": "coheed@mail.com", 
      "Birthday": "1975-03-14T00:00:00.000Z",
      "FavoriteMovies": [] 
    }
Return a list of all Users /users GET None A JSON object holding data about all users. Example: 
              
                [
                {
                  "_id": "62b5176a8ee3f54638b26e4c",
                  "Username": "Coheed",
                  "Password": "the1&only", 
                  "Email": "coheed@mail.com", 
                  "Birthday": "1975-03-14T00:00:00.000Z",
                  "FavoriteMovies": [] 
                },
                ...
                ]
Allow users to update their user info /users/[Username] PUT :Username A JSON object holding the new user information. Example: 
              
    { 
      "Username": "Coh33d",
      "Password": "the0ne&only", 
      "Email": "coh33d@mail.com",
      "Birthday": "1975-03-05T00:00:00.000Z"
    }
A JSON object with the updated user profile. Example: 
              
    { 
      "_id": "62d457b759eb3267c6b240dc",
      "Username": "Coh33d",
      "Password": "the1&only", 
      "Email": "coh33d@mail.com",
      "Birthday": "1975-03-05T00:00:00.000Z",
      "FavoriteMovies": []
    }
Allow users to add a movie to their list of favorites /users/[Username]/movies/[MovieID] POST :Username, :MovieID None A JSON object with the updated user profile. Example: 
              
    { 
      "_id": "62d457b759eb3267c6b240dc",
      "Username": "Coh33d",
      "Password": "the1&only", 
      "Email": "coheed@mail.com",
      "Birthday": "1975-03-04T00:00:00.000Z",
      "FavoriteMovies": [ "62b12273c52bb163a3545262" ]
    }
Allow users to remove a movie from their list of favorites /users/[Username]/movies/[MovieID] DELETE :Username, :MovieID None A JSON object with the updated user profile. Example: 
              
    { 
      "_id": "62d457b759eb3267c6b240dc",
      "Username": "Coh33d",
      "Password": "the1&only", 
      "Email": "coheed@mail.com",
      "Birthday": "1975-03-04T00:00:00.000Z",
      "FavoriteMovies": []
    }
Allow existing users to deregister /users/[Username] DELETE :Username None A message indicating the user was deregistered
